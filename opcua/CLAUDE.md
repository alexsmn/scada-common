# OPC UA

This directory contains the in-repo OPC UA Binary adapter and shared OPC UA server-runtime code.

For the formal OPC UA Binary type schema source of truth, refer to the OPC Foundation schema:

- `https://files.opcfoundation.org/schemas/UA/1.04/Opc.Ua.Types.bsd.xml`

That schema should be treated as the external reference when updating binary message definitions, code generation, or compatibility checks in this directory.
